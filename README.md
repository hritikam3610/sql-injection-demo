# sql-injection-demo
this is  my first project
The project is a SQL Injection Demo application, and it is created using Python, Flask, and SQLite. This application shows a demonstration of a vulnerable login system and a secure login system.

In a vulnerable login system, user data is directly used in a query, and this makes it possible to bypass the system using a SQL Injection attack. However, in a secure system, a parameterized query is used to avoid such an attack.

The project is useful in understanding basic web security concepts and their significance.
